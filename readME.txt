***********|E-VOTING-SYSTEM SETUP|***********

1.) Download or clone this github project (https://github.com/junrek22/CS-e-voting-System/)
2.) In your desktop, Go to C:\xampp\htdocs (If you can't find it, just search where is your xampp installed then to go 'htdocs')
3.) Copy and paste the github project e-voting system file at 'htdocs'
4.) Open and start the Xampp Control Panel - [Start 'Apache' and 'MySQL'] 
    (If you haven't installed it yet, Download Xampp Control Panel)
5.) Go to PhpMyAdmin
6.) Create database and named it "evoting"
7.) Import the existing database provided inside the "database" folder in the github project (e-voting system)
8.) Ready to go

search localhost/e-voting-system/

NOTE - The username and password for the admin page in e-voting-system is [Username: admin , Password : admin]
